# Titanic Data Cleaning & Preprocessing Project

## 📌 Objective

To clean and preprocess the Titanic dataset by handling missing values, removing duplicates, converting data types, and preparing the dataset for analysis.

## 📊 Dataset

Titanic dataset (Kaggle - test file)

## 🛠 Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## 📌 Steps Performed

* Loaded dataset using pandas
* Explored dataset structure using info() and describe()
* Handled missing values (Age, Fare, Embarked)
* Removed duplicate records
* Converted data types (categorical columns)
* Renamed columns for better readability
* Saved cleaned dataset as CSV file

## 📁 Output

* Cleaned dataset: `titanic_cleaned.csv`

## 🎯 Outcome

The dataset is now clean, structured, and ready for exploratory data analysis (EDA) or machine learning models.
